Data Visualizations

JavaScript (D3.js), HTML, CSS

This is a suite of smaller projects; multiple visualizations of different datasets in the form of HTML pages which I created during my Data Visualization course in my final semester at the University of Arizona. Many of the visualizations have various interactions, manipulating the way the data is drawn to empahsize different correlations. Through most, I used the D3.js library to handle accurate data plotting, interactions, and transitions in the visualizations. Some specific data visualization topics I covered through these projects are parallel coordinates, volumetric rendering, field and map visualizations, scatterplots, etc.

Although basic empty templates were often provided by my professor, my JavaScript code handled all tasks that involved creating the displayed visualization from the data. I also used CSS to edit styling and HTML for small layout tweaks. Some interactions include both 1D and 2D brushing over datapoints, interactive 3D object visualization, changing axes, and focus + context.

Relevant Skills

  • HTML: These projects are all developed as HTML pages which serves as a base for JavaScript code which generates the visualizations.
  • CSS: These are styled using CSS (most of the time) to provide more usability and user-friendly appearance.
  • JavaScript: JavaScript code is the foundation used to create almost all of these visualizations.
  • D3.js: D3.js is the JavaScript library used to handle almost all tasks with creating the visualizations. Techniques such as data binds, data joins, and transitions are used to make them work.
  • Data Visualization: These are all visualizations of data, employing data visualization principles in varying degrees such as color, focus + context, expressiveness, and effectiveness.
