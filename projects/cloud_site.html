Cloud Mics Website

Shopify

This was a Shopify-based website which I designed in a team of 3 during my time working for Cloud Microphones' marketing team. Although mostly put together using the Shopify editor, there were several instances where I had to modify JavaScript, CSS, HTML, and Shopify Liquid code to surpass the capabilities of the given theme. This was my first professional project outside of an educational environment, where I had to employ an e-commerce mindset when designing product pages and prioritizing usability.

Some of the parts of the website that I worked on, apart from contributing to the general layout of each page, included the animations and interactions of different slides on the home page, the AI chatbot window accessible from the top right, the dynamic nature of the Artists page (new artists are easily dynamically added to the page using Shopify metaobjects), and custom forms throughout different pages.

Relevant Skills

  • Shopify: The website was built with Shopify, using the editor and theme templates to provide the basic layout of the site, whenever possible.
  • Liquid: Liquid is the HTML-based language that Shopify runs with, which I edited in the case of the Shopify editor not providing enough capability.
  • HTML: In editing liquid code, I used HTML to develop new components and pages.
  • CSS: I used CSS to tweak existing styles to ensure design consistency with existing and new components of the website.
  • JavaScript: JavaScript was used to handle some functions for introduced snippets.
  • Collaborative: This project was my first collaboration in a non-educational environment; I worked in a team of 3 closely to ensure consistency.
  • E-Commerce: I applied an e-commerce mindset to the layout and design of the website.
