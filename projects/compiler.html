C to MIPS Compiler

This project was a compiler I wrote over the course of a semester as part of my Compilers course (CSC 453) at the University of Arizona. Written completely in C using a Linux CLI (Vim editor), it translates a programming language dubbed "C--" (a significant subset of C functionality) into functionally identical MIPS assembly code. The full spec of the C-- language can be found here. In essence, this successfully compiles function calls, for loops, conditionals, variable declarations, and arithmetic.

This project was developed through a pipeline of stages throughout the semester. I learned the principles of compilation, starting with lexical analysis, then semantic analysis through symbol table management. We covered intermediate represenatations, specifically abstract systax trees and three-address code. Finally, we translated into completed MIPS code, ensuring functionality using the SPIM command-line tool.

Relevant Skills

  • C: Coded entirely in C using the Vim text editor on Linux. It also required a thorough understanding of the behind-the scenes functionality of C, as it was the source language of compilation.
  • MIPS Assembly: A thorough understanding of MIPS assembly was required to replicate the functionality of primitive C operations.
  • UNIX/Linux: I used Linux as my primary development environment for this project. I used CLI tools such as SPIM and GCC to run C and MIPS code.
  • Vim: I used Vim as the text editor for the entire project.
  • Compilation: I employed knowledge about the foundational steps of compilation while making this project.
