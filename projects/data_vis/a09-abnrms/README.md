## Notes

This submission draws four plots visualizing contours in the Hurricane Isabel dataset. Two plots are outline contours of the temperature and pressure, and the other two are filled contours of the same data.

To see this visualization, simply open index.html in a modern browser (I use Chrome) while all files are in the same directory.

I did not pursue any extra credit for this assignment.

## Included files

* README.md - This file
* index.html - The html file to open, loads all .js files
* a09.js - The code to generate the visualization, uses the d3 library and the hurricane data
* d3.js - The library used by a09.js
* data.js - The Hurricane Isabel data, encoded into a grid of cells to be used in a09.js.
