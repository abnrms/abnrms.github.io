## Notes

This repository implements a treemap for the Flare dataset provided by Jeff Heer. It includes four buttons on the top right for interacting with the vis. You can decide whether to display the size of the nodes by their child count or size in the dataset, and you can choose whether to alternate which direction the rectangles are drawn or choose based on their parent's dimension.

To view this visualization, simply open index.html in a modern browser (I use Chrome) while all files are in the same directory.

I did not pursue any extra credit for this assignment.

## Included files

* README.md - This file
* index.html - The file to open to view the visualization. Loads all .js files in the directory.
