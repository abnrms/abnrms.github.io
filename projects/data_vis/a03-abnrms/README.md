## Notes

To run this program, simply open index.html in a modern browser (I used Chrome), while all the included files are in the same directory. You should see a beautiful assembly of the same images we were asked to reproduce.

I decided not to pursue any extra credit in this assignment. I originally had used scales to reproduce the images while making them look cleaner (even spacing and no cutoffs on the images) but I took that AWAY! See my commit history if it matters.

## Included files

* README.md - This file
* index.html - The html page which includes and renders the rest
* a03.js - The javascript file which generates the four plots
* style.css - The stylesheet for the html file
* d3.js - The d3 library used in a03.js
* calvinScores.js & ukDriverFatalities.js - The datasets used in a03.js
